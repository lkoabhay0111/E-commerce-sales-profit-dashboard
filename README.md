# E-commerce Sales & Profit Dashboard

## Project Overview
This project analyzes e-commerce sales data to understand sales performance,
profitability, and trends. Data cleaning and analysis were performed using
Python (Jupyter Notebook), and an interactive dashboard was built in Power BI
using Python script integration.

## Tools Used
- Python (Jupyter Notebook)
- Pandas, Matplotlib
- Power BI
- DAX

## Key Metrics
- Total Sales: 2.30M
- Total Profit: 286.40K
- Total Orders: 5K
- Profit Margin: 12.47%

## Analysis Performed
- Monthly sales and profit analysis
- Category and sub-category performance
- Region-wise profit analysis
- Customer segment analysis
- Sales-to-profit ratio calculation
- Null and duplicate value handling

## Dashboard Features
- KPI cards for sales, profit, orders, and margin
- Monthly sales and profit trends
- Category, sub-category, and region analysis
- Interactive slicers (Date, Category, Region)

## Files in Repository
- Power BI Dashboard 
- Project Presentation 
- Cleaned Dataset
- jupyter notebook file(Data cleaning, analysis, and visualization in Python)

## Conclusion
The dashboard provides actionable insights into sales performance and
profitability, helping stakeholders make data-driven decisions.
